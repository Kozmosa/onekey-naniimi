# onekey-naniimi

何意味转换器 - 一个有趣的中文文本转换工具

## 功能介绍

这是一个单页面纯静态应用，可以将中文文本中的字词随机替换为"何"、"意"、"味"的组合，产生有趣的文本效果。

### 转换规则

- **单字**：随机替换为"何"、"意"或"味"中的任意一个
- **双字词**：替换为"意味"
- **三字词**：替换为"何意味"

所有替换都是基于可配置的概率进行的，默认概率为50%。

### 主要特性

✨ **美观的界面设计**
- 现代化的渐变背景
- 卡片式布局
- 流畅的动画效果

🎯 **智能分词**
- 基于字典的中文分词
- 支持常见的2字词和3字词识别
- 保留标点符号和非中文字符

⚙️ **可配置的设置**
- 可调节单字替换概率（0-100%）
- 可调节双字词替换概率（0-100%）
- 可调节三字词替换概率（0-100%）

📱 **响应式设计**
- 支持桌面和移动设备
- 自适应布局

## 使用方法

1. 打开 `index.html` 文件（可以直接在浏览器中打开，无需服务器）
2. 在"输入文本"框中输入需要转换的中文文本
3. 点击"转换"按钮查看结果
4. 点击"设置"按钮可以调整各种替换的概率

### 示例

输入：
```
我是中国人，你好世界！今天天气真好。
```

输出（示例，因为是随机的，每次结果可能不同）：
```
我味何意味，意味世界！意味天气真好。
```

## 开发

这是一个纯静态应用，无需构建步骤。直接在浏览器中打开 `index.html` 即可运行。

如果需要本地开发，可以使用任何静态文件服务器，例如：

```bash
# Python 3
python3 -m http.server 8080

# Node.js
npx serve

# PHP
php -S localhost:8080
```
